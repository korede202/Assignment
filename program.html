Programming Language

A Programming language is a system of natation for writing programs.programming languages and described in terms of their syntax and semantics,usually defined by a formal language.An implementation of a programming language in the form of a compiler or interpreter allows programs to be excuted,either directly or by producing what is known in programming as an executable.

5 major types of programming language

  • Functional Programming language
  • Object-oriented programming (OOP)
  • Scripting language
  • Logic programming language
  • Procedural Programming language.A procedural language follows a sequence of statement or commands in order to achieve a desired output
